# 여행은 계속 된다

## 23.2 함수형 사고

> 효과란, 어떤 일이 발생하는 것

- 효과가 적게 발생하는 코드가 좋은 코드
- 함수 안에서 컬렉션이나 변수를 변경하는 일은 효과
- 변경한 내용이 함수 밖으로 공유되지 않으면 효과 x &rarr; 변경한 내용이 함수 밖으로 영향 x

#### 간단한 설계
